# 🇮🇳 The Story

Every year, Members of Parliament receive **₹5 Crore** through the MPLADS scheme to improve their constituencies.

But one question remains unanswered.

> **Which development project should be funded first?**

Today, citizens report issues through

📞 Phone Calls

📱 WhatsApp

📄 Letters

🏛 Public Meetings

🌐 Government Portals

The result is fragmented information, duplicate complaints, and invisible communities.

Development decisions often depend on **who speaks the loudest**, instead of **where the need is greatest**.

We believed India deserved better.

So we built **JanPriority**.
